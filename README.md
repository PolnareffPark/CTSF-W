## CTSF-W 는 W1 ~ W5 실험을 진행하기 위한 레포입니다.

각 실험에 관한 구체적인 설명은 exp-plan 폴더 안에 들어 있는 각각의 CTSF-W@-plan.md 파일과 CTSF-W-Overview.md 를 참고하십시오.

기존 실험에서, 본 실험의 contribution, novelty 방향을 바꿈에 따라 기존 Ablation Study 결과를 사용할 수 없습니다.

이에 새로운 Ablation Study 를 진행하는 것이 필요하기 때문에 이 레포가 만들어졌습니다.

기존의 연구 결과에 관한 정보는 첨부한 PDF 를 참조합니다.

## git commit 작성 규칙

본 레포의 커밋은 다음과 같은 규칙을 사용합니다

[ADD] : 새로운 파일이 추가되었을 때 add 합니다. 파일 변경과 추가가 동시에 이루어진 경우 [ADD] 로 작성합니다.

[REMOVE] : 기존 파일을 지웁니다.

[UPDATE] : 기존 파일을 변경합니다.

[CHORE] : 코드에 영향을 주지 않고, 파일의 위치를 변경하는 등의 경우 사용합니다.

[EXCEPT] : 등록된 규칙 외에 다른 커맨드가 있을 경우 사용합니다.

[MERGE] : 브랜치를 병합합니다.
